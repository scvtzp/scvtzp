# Hi there 👋
### 💬 About Me  
안녕하세요!
게임 클라이언트 프로그래머 남대현입니다.
가장 좋아하기도 하고 개발에 관심이 많은 장르는 턴제 전략 게임(세븐나이츠, 하스스톤, 문명, 오토체스)입니다.
게임플레이 구현에 관심이 많고, 그 외에도 알고리즘, 최적화, 그래픽스에 관심이 있습니다.
